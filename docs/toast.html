Toast

Use <output popover="manual"> for action feedback. Inverted fg/bg, slides up from the bottom. Requires a small JS helper to show and auto-dismiss.

Demo

Markup

<button type="button" id="toast-trigger">Show toast</button>
<output id="toast" popover="manual" role="status" aria-live="polite">
  Saved.
</output>
toast.showPopover();
setTimeout(() => toast.hidePopover(), 3000);
← Card Nav →
Action completed — toast dismissed automatically.
