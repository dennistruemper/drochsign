Text input

Text inputs use an inset border inside a reserved 2px box — focus thickens the border without layout shift. Errors use aria-invalid and a dashed border.

Demo

Only letters are allowed — no numbers or symbols.

Type a number or symbol to trigger the error state.

Markup

<label for="name">Username</label>
<input id="name" type="text" aria-describedby="name-error" aria-invalid="true" />
<small id="name-error" role="alert">Only letters allowed.</small>
